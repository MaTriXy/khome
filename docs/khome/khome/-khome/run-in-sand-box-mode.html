runInSandBoxMode

fun runInSandBoxMode(action: () -> Unit): Unit

Call some action in an sand box mode. The sand box mode allows you to act like you would call the hass websocket api but without actually calling it, by using khome.calling.callService. Use this to do some testing or playing around.

The call service payload will be printed to the logs.
